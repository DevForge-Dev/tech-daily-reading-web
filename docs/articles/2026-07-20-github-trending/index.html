7/20の話題リポジトリ — エージェント運用の土台（コスト削減・永続記憶・隔離実行）

今日の3本を貫くのは「多数のエージェント/定期タスクを回す現場の“高い・忘れる・危ない”に答える」こと。コストを削るゲートウェイ、記憶を永続化するメモリ基盤、安全に実行するサンドボックス。

📅 2026年7月20日 ⏱ 約2分 🔗 出典 3 件

今日GitHubで急上昇した中から、エージェント運用の土台を成す3本を取り上げる。コストを削るゲートウェイ、記憶を永続化するメモリ基盤、安全に実行するサンドボックス——多数のエージェントや定期タスクを日々回す現場が直面する「高い・忘れる・危ない」に、それぞれ正面から答える顔ぶれだ。

diegosouzapw/OmniRoute（TypeScript・⭐19k）

解決したい課題：多数のモデル/プロバイダを使い分けるほど、接続先の分散・トークン浪費・障害時の全停止がコストと不安定さを生む。提供する機能：231以上のプロバイダ（50以上は無料）を1エンドポイントに集約し、Claude Code/Codex/Cursor/Cline/Copilotから各種モデルへ接続。RTK+Caveman方式のスタック圧縮でトークンを15〜95%削減し、障害時はスマート・フォールバックで自動切替。MCP/A2A対応で既存のtool-use運用に載せやすい。findarepoで直近7日+約3.3kスターと急伸中で、コスト削減文化に直撃している。

topoteretes/cognee（Python）

解決したい課題：エージェントはセッションを跨ぐと文脈を忘れ、毎回同じ前提を読み直してトークンを浪費する。提供する機能：会話・作業のたびに知識をナレッジグラフへ蓄積し、過去セッションを参照できる永続的な長期記憶を、セルフホストで与えるAIメモリ基盤。毎回のコンテキスト再読込を減らせるため入力トークン削減に直結する。Trendshift日次13位と着実に上昇し、機微データを外へ出さない運用とも整合する。

TencentCloud/CubeSandbox（Rust）

解決したい課題：エージェントが生成したコード/コマンドをそのまま実行するのは破壊的操作のリスクがあり、隔離が要る。提供する機能：即時起動・並行実行・安全・軽量なサンドボックス実行環境をRust製で提供し、生成コードを隔離環境で高速に実行する。OSSInsightの当日トレンドに登場、大手Tencent発で、多数の定期タスクを回す運用の安全網に適合する。

まとめ

自分が試すなら、まずOmniRoute（即コスト効果）から。記憶のcogneeと隔離実行のCubeSandboxを足せば、コスト・記憶・安全の三点が揃う。
