今日の急上昇3本を貫くのは、「著名開発者が自分の得意領域を丸ごとOSSで開く」流れだ。UIの名手はスキルを、Spacedriveの作者は音声を、WebのVercelはZig製ネイティブを——それぞれコード生成AIが崩しがちな領域やWebの外側を補いにいっている。

emilkowalski/skills（スキル集・OSSInsight日次3位）

解決したい課題：コード生成AIはアニメーションやインタラクション設計といった“質感”の領域を崩しがちで、実務のデザイン勘が抜け落ちる。提供する機能：Sonner/Vaul で知られる著名デザインエンジニア Emil Kowalski が、デザインエンジニアの実務知をAIエージェント用スキル集として公開。続くスキル・ブームの中でも著名開発者発の信頼感で拡散が速く、本日New で日次3位に急浮上した。

jamiepine/voicebox（TypeScript・日次上位）

解決したい課題：高品質な声クローン・音声生成は ElevenLabs 的なクローズド／従量課金サービスに依存しがち。提供する機能：声のクローン・ディクテーション（音声入力）・音声生成を1つに統合したオープンソースの AIボイススタジオで、自前ホストできる。Spacedrive のファウンダー Jamie Pine の新作という話題性も後押しし、AIエージェント一色の中で数少ない「音声クリエイティブ」枠として浮上した。

vercel-labs/native（Zig・日次ランクイン）

解決したい課題：ネイティブなデスクトップアプリの選択肢は Electron/Tauri に偏り、より低レイヤで軽量な土台が乏しい。提供する機能：Zig で実装されたネイティブデスクトップ・ツールキット。Web 領域で知られる Vercel の Labs がWeb外の低レイヤに踏み込んだ意外性と、AIツールが並ぶトレンドでの言語多様性（Zig）で注目を集めている。
