safishamsi/graphify（Python・⭐76.9k / 本日+約937）

解決したい課題: AI エージェントにコードベースを渡しても、アプリコード・SQL スキーマ・インフラ構成・ドキュメント・論文・画像・動画といった資産はバラバラのファイル群のままで、「この API はどのテーブルに触り、どの設計文書に根拠があるか」といった横断的な文脈を機械が把握できない。提供する機能: 任意のフォルダを問い合わせ可能なナレッジグラフに変換する AI コーディング支援スキル。Claude Code・Codex・OpenCode・Cursor・Gemini CLI など主要エージェントから「スキル」として使え、コードと DB スキーマとインフラを1つのグラフとして横断クエリできる。1日で約940スターという急伸は、特定ツールに縛られないスキル設計が刺さった証拠だろう。

ChromeDevTools/chrome-devtools-mcp（TypeScript・⭐45.4k / 本日+約404）

解決したい課題: コーディングエージェントは「コードを書く」ことはできても、書いたものがブラウザでどう振る舞うか — 描画性能、ネットワーク、コンソールエラー — を自分で確かめる手段を持たない。提供する機能: Chrome DevTools の機能を MCP サーバーとしてエージェントに公開し、パフォーマンス計測・ネットワーク検査・デバッグ情報へのプログラム的アクセスを可能にする。Chrome DevTools チーム公式（ChromeDevTools org）という信頼性が採用の後押しで、MCP 対応ツールの需要拡大を背景に着実に伸びている。

rommapp/romm（Python・⭐9.7k / 本日+約236）

解決したい課題: レトロゲームの ROM コレクションはフォルダに散らばりがちで、メタデータ整理もプレイ環境の準備も手作業になりやすい。提供する機能: セルフホスト型の ROM マネージャー兼プレイヤー。ライブラリのメタデータを自動取得して整理し、ブラウザから直接プレイでき、Docker で手軽に立てられる。AI 一色のトレンドで異色の存在だが、セルフホスト愛好家コミュニティの支持で安定して上昇中だ。

まとめ: 自分が試すなら chrome-devtools-mcp。Claude Code に繋いで「実装 → ブラウザで計測 → 直す」のループをエージェントに閉じさせられるかを確かめたい。graphify はモノレポ＋DB スキーマの案件で真価が出そうだ。
