今日の3本を貫くもの

本日のTrendingは、AIエージェントを実務に組み込む前提での土台づくりという共通線で読める。エージェントに能力（スキル）を与える枠組み、エージェントが扱いやすいUIの器、そしてモデルではなくシステム全体を理解するための知識——道具・UI・学びの3方向から地ならしが進む。それぞれ「何を解決しようとしているか」「何を提供するか」で見ていく。

obra/superpowers（Shell・⭐245.4k）

解決したい課題：コーディングエージェントは強力だが、都度その場のプロンプトに頼ると再現性がなく、ノウハウが蓄積しない。提供する機能：エージェントに再利用可能な「スキル」を装備させ、開発メソドロジーとツールを一体で提供する枠組み。Claude Code などのエージェントと組み合わせやすい。本日約1,200スターを獲得し、すでに24万超という規模から、AIエージェント開発の実践知として拡散していることがうかがえる。

facebook/astryx（TypeScript・⭐4.4k）

解決したい課題：既存のUIコンポーネント群は人間の手作業前提で、エージェントが安全にカスタマイズ・生成しづらい。提供する機能：完全カスタマイズ可能で “agent ready” を掲げる、Meta製OSSのデザインシステム。AIによるUI構築の生産性向上を狙う。大手Metaが公開した信頼性と、エージェント対応という新規性が本日900超スターの急伸を後押ししている。

harvard-edge/cs249r_book（Python・⭐26.1k）

解決したい課題：学習リソースはモデル単体に偏りがちで、データ・運用・システム設計まで含む全体像が学びにくい。提供する機能：ハーバード発、機械学習システム（ML Systems）を体系的に学べるオープンな教科書。モデル運用やシステム設計まで幅広くカバーする。大学発の無料教材という信頼性と網羅性から、本日約800スターを得て教育リソースとして広がっている。
