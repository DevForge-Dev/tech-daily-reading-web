型を付けるとグラデが動く — @propertyでCSSだけの回転ボーダーと色溶け

カスタムプロパティは既定で“文字列”なので補間できず、--angleを0→360degにしてもカクッと切り替わるだけだった。@propertyで角度/色/数値と型を宣言すると、変数そのものが補間可能になり、conicの回転ボーダーや色停止の溶け込み、進捗リングが純CSSで書ける。Baseline 2024で主要3エンジン対応、RSCのまま use client 不要。

📅 2026年7月7日 ⏱ 約4分 🔗 出典 4 件

なぜ今これが話題か

CSS変数（--x）は既定ですべて文字列扱いのため、--angle: 0deg → 360deg と書いても補間されずカクッと切り替わるだけだった。グラデーションの角度や色停止は値の内部にあり、transition/@keyframesの対象にできない。だから従来は「要素ごとtransform: rotate()」「JSでrAFループ」「重ね画像をクロスフェード」で回避していた。@propertyはこの前提を覆す。

何が新しいか

@propertyはカスタムプロパティに型（syntax）を宣言する。型が付くと、ブラウザはその値を補間可能な<angle>/<color>/<number>等として理解し、変数自体をアニメーションできる。対応はBaseline 2024（2024-07-09 Newly available）で、Chrome/Edge 85+、Safari 16.4+、Firefox 128+と主要3エンジンが揃い、実務投入OKの段階だ。フォールバックは「動かない静的グラデ」に自然に縮退するので、加点演出に向く。

仕組み・要点

宣言は3ディスクリプタ。syntaxとinheritsは常に必須、initial-valueはsyntax:"*"以外で必須（＝フォールバック値）。型に合わない代入はinitial-valueに落ちるので型安全だ。

@property --angle {
  syntax: "<angle>"; inherits: false; initial-value: 0deg;
}
.spinner {
  background: conic-gradient(from var(--angle), #6574cd, #3490dc, #ffb636, #6574cd);
  animation: spin 1.2s linear infinite;
}
@keyframes spin { to { --angle: 360deg; } }

回転ボーダーは擬似要素なしで作れる：回転するconicを背景に敷き、padding分だけ内側にソリッド背景の子を重ねると隙間が枠に見える。中身は回らず枠だけ回るのが利点。色溶けは各色停止を別々の@property <color>にしてvar()参照し、transition: --c1 .6sのように変数を直接トランジションする（素のグラデ同士のtransitionは効かない）。<number>を使えばconic-gradient(#22c55e calc(var(--p)*1%), …)で進捗リングも動かせる。

Next.js（App Router）での使いどころ

純CSS＝Server Componentのままで、@propertyも@keyframesもクライアントJSを要さない。ローダー、CTAの光る枠、ヒーロー背景をRSCで完結できhydrationゼロ・INP/LCP/CLSに悪影響なし。@propertyはapp/globals.cssに一度グローバル登録すれば全ページで使える。サーバー値で決めたい進捗はインラインの--p変数でRSCが出力、カウントアップ等の動的変化だけ小さな'use client'島に閉じる。Tailwind v4の@themeトークンをinitial-valueへ流用も可。

注意点・落とし穴

initial-value忘れは登録無効化＝補間が効かない。色停止は停止ごとに型付き変数が必要（1変数に複数色は不可）。inherits:trueの乱用で子孫が意図せずアニメを拾う。大面積conicの毎フレーム再ペイントは重く、領域は小さめ＋contain: paintを検討。JSのCSS.registerProperty()はSSR/RSCで実行されないので、SSRから確実に効かせたいならCSSの@propertyを使う。@media (prefers-reduced-motion: reduce)でanimation: noneにすれば静的グラデの完成形が残る。

ひとことまとめ

「型を付ける」だけでグラデが動く。回転ボーダー・色溶け・進捗リングをJSも擬似要素もなしにRSCで完結できる、コスパの高い装飾テクだ。
