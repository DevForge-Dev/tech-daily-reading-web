Flutter 2026 の旗印「Platform Parity」と GenUI — A2UI でAIが実ウィジェットを動的構成、HCPP で Platform Views を刷新

安定版は Flutter 3.44.0 / Dart 3.12.2（Google I/O 2026）。2026 の旗印は「Platform Parity（プラットフォーム同等性）」で、ネイティブに近いが届かない uncanny valley の解消を狙う。目玉は GenUI SDK で、オープンな A2UI（Agent-to-UI）プロトコル上に構築され、AIエージェントが実 Flutter ウィジェットをその場で構成して静的ウィジェットツリーの制約を突破する。Android の Platform Views は Hybrid Composition++（HCPP）で刷新された。

📅 2026年6月28日 ⏱ 約4分 🔗 出典 3 件

なぜ今これが話題か

Flutter は2026年、「Platform Parity（プラットフォーム同等性）」を旗印に掲げた。アプリがネイティブに「近いが届かない」uncanny valley の解消が狙いで、レンダリングからプラットフォーム連携まで底上げが進む。安定版は Flutter 3.44.0 / Dart 3.12.2（Google I/O 2026, 2026-05-18）。次の安定版は2026年8月予定で、年4回（2/5/8/11月）のリズムは変わらない。

何が新しいか

今日の主役は GenUI SDK と HCPP。GenUI はオープンな A2UI（Agent-to-UI）プロトコル上に構築され、AIエージェントが実 Flutter ウィジェットをその場で構成し、静的ウィジェットツリーの制約を突破して、ユーザーの目的に応じた完全インタラクティブな画面を動的生成する。Android の Platform Views は Hybrid Composition++（HCPP）で刷新され、Vulkan ＋ SurfaceControl でコンポジットを Android OS へ委譲することで、長年の「性能 vs 正確性」のトレードオフを解消した。iOS/macOS は SwiftPM が既定に。Dart 3.12.2 は private named parameters・実験的 primary constructors・dot shorthands で宣言を簡潔化し、Agentic Hot Reload と Genkit でAI/エージェント開発を標準装備した。採用例は 2026 Toyota RAV4、LG webOS SDK for Flutter が近日対応。

足回り：レンダリングと状態管理

描画は Impeller が既定（Skia 置換）。シェーダーをビルド時(AOT)に事前コンパイルして初回ジャンクを解消し、複雑シーンの平均ラスタライズ時間を約50%短縮、高リフレッシュ端末で 120fps を維持する。状態管理は新規なら Riverpod 3.0 が既定（コンパイル時安全・組込みオフライン永続化・最小ボイラープレート、初期化失敗時は 200ms→倍々で最大6.4s の指数バックオフで自動リトライ）。監査証跡が要る規制業界（金融/医療）は完全な event traceability を持つ Bloc 9.0 が標準で、v9.0 の mounted 安全チェックが disposed widget へのコールバック実行を防ぐ。

自分が試すなら

  • Platform Views を多用する画面で HCPP（Vulkan＋SurfaceControl）の効果を、地図やWebView混在のスクロール性能で確認する。
  • 新規の小さな画面で GenUI / A2UI の動的UI生成を試し、静的ツリーで書く場合との差を体感する。
  • Dart 3.12 の dot shorthands や primary constructors（実験的）で、enum/定数まわりの宣言がどれだけ短くなるか試す。

注意点

GenUI / A2UI と HCPP は2026の新しい柱で、APIや既定動作が今後の安定版（8月予定）で変わりうる。primary constructors は実験フラグ段階。Rive×Impeller で描画や性能に差が出る場合は --no-enable-impeller で Skia にフォールバックして切り分ける。パフォーマンスの最頻出問題は依然として不要な再ビルドで、不変部分は const で完全スキップ、大きな build() は分割、計測は必ず profile モードで（フレーム予算 60fps=16ms / 120fps=8.3ms）。

ひとことまとめ

2026の Flutter は「Platform Parity」を軸に、GenUI/A2UI でUIの作り方そのものを動的化し、HCPP で Platform Views の長年の弱点を解消する。足回りは Impeller の120fpsと Riverpod 3.0/Bloc 9.0 の使い分けが安定線。新しい柱は積極的に触りつつ、安定版での変化に備えておきたい。
